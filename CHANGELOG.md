# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-27

### Added
- Initial release
- Support for 6 AI platforms: Claude, OpenAI, DeepSeek, Qwen, Zhipu AI, and Spark
- Generate titles, tags, categories, and summaries using AI
- Preview and edit metadata before applying
- Context menu integration (file explorer and editor)
- Ribbon icon for quick access
- Tag reuse from existing Vault tags
- Category suggestions based on folder structure
- Individual configuration for each AI platform
