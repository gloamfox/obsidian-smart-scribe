# SmartScribe - Obsidian AI 智能助手

使用 AI 智能生成 Obsidian 笔记的标题、标签、分类和摘要，以及优化写作内容的插件。支持多种 AI 平台。

> **插件 ID**: `smartscribe`

## 支持的 AI 平台

| 平台 | 名称 | 默认模型 |
|------|------|----------|
| Claude | Anthropic Claude | claude-sonnet-4-6 |
| OpenAI | OpenAI GPT | gpt-4o |
| DeepSeek | DeepSeek | deepseek-chat |
| 千问 | 阿里通义千问 | qwen-plus |
| 智谱 AI | Zhipu GLM | glm-4 |
| 讯飞星火 | Spark | pro |

## 功能特点

### AI 笔记属性生成
- 🤖 **多平台支持**：支持 6 种主流 AI 平台，可自由切换
- 🏷️ **标签复用**：优先使用 Vault 中已有的标签，保持一致性
- 📂 **分类建议**：支持基于文件夹结构的分类推荐
- ✏️ **预览编辑**：生成后可预览和编辑笔记属性再应用
- 🖱️ **右键菜单**：在文件浏览器和编辑器中右键访问

### AI 文本优化
- ✨ **智能写作助手**：一键优化选中文本或全文
- 📝 **智能改进**：改进表达、结构、语法和可读性
- 🎯 **双栏预览**：左栏展示优化后文本（可编辑），右栏展示改进建议
- 🔄 **直接应用**：可直接应用或先预览再应用

### 灵活配置
- ⚙️ **独立配置**：每个平台独立配置 API Key、模型、参数
- 🔧 **细粒度控制**：笔记属性生成和文本优化功能可分别设置

## 安装

### 手动安装

1. 从 GitHub 下载最新版本
2. 解压文件到你的 Vault 的 `.obsidian/plugins/smartscribe/` 文件夹
3. 在 Obsidian 设置中启用插件


## 配置

1. 打开插件设置
2. 选择要使用的 **AI 平台**
3. 配置对应平台的 **API Key**
4. 选择要使用的**模型**
5. 根据需要调整 Temperature、Max Tokens 等参数
6. 配置功能选项（最大标签数、文件夹结构、预览设置等）

### 各平台 API Key 获取

- **Claude**: https://console.anthropic.com/
- **OpenAI**: https://platform.openai.com/
- **DeepSeek**: https://platform.deepseek.com/

- **智谱 AI**: https://open.bigmodel.cn/
- **讯飞星火**: https://xinghuo.xfyun.cn/

## 使用

### 笔记属性生成

#### 命令面板

- **给当前文章生成笔记属性**：直接生成并应用笔记属性
- **给当前文章生成笔记属性（预览）**：生成后显示预览，可编辑后再应用

#### 右键菜单

在文件列表中右键点击 Markdown 文件，选择 "AI 生成笔记属性"

#### 编辑器菜单

在编辑器中右键，可访问 AI 生成笔记属性选项

### 文本优化

#### 命令面板

- **AI 优化文本**：优化选中文本（未选中则优化全文）
- **AI 优化文本（预览）**：始终先显示预览再应用

#### 右键菜单

在编辑器中右键，选择 "AI 优化文本" 来优化选中文本或全文

#### 使用流程

1. **选择文本**：在编辑器中选中文本，或不选以优化全文
2. **触发优化**：使用命令面板或右键菜单
3. **预览（可选）**：如果开启预览，将显示双栏界面：
   - **左栏**：优化后的文本（可编辑，支持格式）
   - **右栏**：改进建议（分条展示）
4. **应用**：点击"应用"将原文替换为优化后的文本

## 工作原理

### 笔记属性生成
1. **分析阶段**：插件使用 Obsidian 的 MetadataCache API 获取 Vault 中的已有标签
2. **生成阶段**：调用选定的 AI 平台 API 分析文章内容，传入已有的标签和分类作为参考
3. **应用阶段**：将生成的笔记属性写入文件的 YAML frontmatter

### 文本优化
1. **内容捕获**：捕获选中文本或全文内容
2. **AI 处理**：将内容发送给 AI 并附带优化指令
3. **结果解析**：分离优化后的文本和改进建议
4. **预览/应用**：显示预览（如启用）或直接应用优化后的文本

## 设置选项

### 模型配置
- **AI 平台**：选择 6 个支持的平台之一
- **API Key**：所选平台的 API 密钥
- **模型**：选择要使用的模型
- **Base URL**：可选，用于自定义 API 端点或代理
- **Temperature**：生成随机性（0-2，越低越确定）
- **Max Tokens**：最大生成令牌数

### 笔记属性生成设置
- **最大标签数**：生成时参考的已有标签最大数量（1-10）
- **包含文件夹结构**：将文件所在文件夹作为分类建议
- **显示右键菜单**：在右键菜单中显示笔记属性选项
- **生成前显示预览**：生成笔记属性前显示预览窗口

### 文本优化设置
- **显示优化菜单**：在编辑器右键菜单中显示"AI 优化文本"选项
- **优化前显示预览**：优化文本前显示预览窗口

## 注意事项

- 需要有效的 API Key，各平台单独配置
- API 调用会产生费用，请留意使用量
- 建议在生成前备份重要文件
- 可通过 Temperature 参数调整生成结果的创造性
- 文本优化功能对较短的文本效果可能不明显（建议至少几句话）
